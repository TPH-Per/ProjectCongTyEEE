## 3. Fix kéo thả khách vào bàn (drag & drop)

### Mô tả

Khi kéo khách từ danh sách chờ vào bàn, bàn không được cập nhật thông tin khách.

### Phân tích nguyên nhân (Root Cause)

`simulatedAreas` là một `computed` quyết định trạng thái bàn dựa trên vị trí timeline slider so với giờ đặt bàn. Khi drop khách vào bàn:

1. `handleDrop()` gán khách vào bàn qua `receptionAssignTable()` — store cập nhật đúng ✅
2. Nhưng `simulatedAreas` re-compute — và nếu timeline slider không khớp giờ đặt bàn (ví dụ: timeline đang ở 11:00, khách đặt 18:30), vòng lặp timeline **không set `foundActive = true`**
3. Kết quả: bàn **revert về `Available`** không có khách — ghi đè assignment

### Files sửa

| File | Hành động | Mô tả |
|------|-----------|-------|
| `src/views/reception/ReceptionFloorsView.vue` | **Sửa** | Fix 3 điểm trong `handleDrop` + `simulatedAreas` |

### Chi tiết fix

#### Fix 1: `simulatedAreas` — fallback cho bàn đã gán khách (dòng ~1210)

```typescript
// Trước: nếu timeline không match → return Available (mất khách)
// Sau: nếu tableBookings.length > 0 nhưng timeline không match → vẫn hiển thị Reserved
if (tableBookings.length > 0) {
  return { ...table, status: 'Reserved', customerName: tableBookings[0].customerName, billAmount: '', checkInTime: '', occupiedDuration: '' }
}
return { ...table, status: 'Available', customerName: '', billAmount: '', checkInTime: '', occupiedDuration: '' }
```

#### Fix 2: `handleDrop` — kiểm tra sức chứa bàn (dòng ~1045)

```typescript
// Thêm capacity validation trước khi gán
const totalGuests = reservation.adults + reservation.children
if (totalGuests > table.capacity) {
  // Shake animation + Swal warning toast
  return
}
```

#### Fix 3: `handleDrop` — success toast ngay lập tức (dòng ~1085)

```typescript
// Trước: success toast chỉ fire khi Supabase .update() thành công
// (không chạy cho mock data hoặc khi offline)
// Sau: fire Swal success toast ngay sau khi receptionAssignTable()

Swal.fire({
  toast: true,
  position: 'top-end',
  showConfirmButton: false,
  timer: 2000,
  timerProgressBar: true,
  icon: 'success',
  title: t('reception_floors.assign_success'),
})
```

Loại bỏ Supabase `.update()` call vì `receptionAssignTable` + `persistToStorage()` đã xử lý persist.
